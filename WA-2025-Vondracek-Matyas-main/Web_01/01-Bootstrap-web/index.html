Tlačítka

Bootstrap umožňuje snadné stylování tlačítek pomocí tříd jako např. button-primary.

            
            <button class="btn btn-primary btn-lg">Primary button</button>
            <button class="btn btn-warning">Warning button</button>
            <button class="btn btn-danger btn-sm">Danger button</button>

Průhledná tlačítka

            
                <button type="button" class="btn btn-outline-success">Success</button>
                <button type="button" class="btn btn-outline-primary">Primary</button>
                <button type="button" class="btn btn-outline-danger">Danger</button>

Karty

Karta v bootstrapu je skvělý způsob jak strukturovat obsah.

Ukázková Karta

Toto je ukázková karta v bootstrapu.

Detail

            <div class="card-body">
                <h3 class="card-title">Ukázková Karta</h3>
                <p class="card-text">Toto je ukázková karta v bootstrapu.</p>
                <a href="#" class="btn btn-info">Detail</a>
            </div>
        </div>
